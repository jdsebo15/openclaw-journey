# OpenClaw Journey Blog

A blog documenting my journey learning OpenClaw, Claude Code, and building apps with AI assistants.

## About This Blog

This blog focuses on practical, hands-on experience with AI coding assistants:

- **🤖 OpenClaw** - Open-source AI assistant framework
- **💻 Claude Code** - AI coding assistant from Anthropic
- **🚀 App Development** - Building real applications with AI help
- **💰 Side Business** - Turning AI-assisted development into income

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Content**: Static pages with planned MDX integration
- **Deployment**: Vercel (planned)
- **Domain**: TBD

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
openclaw-journey/
├── app/                    # Next.js app router
│   ├── blog/              # Blog routes
│   │   └── [slug]/        # Individual blog posts
│   ├── layout.tsx         # Root layout
│   └── page.tsx           # Homepage
├── components/            # React components
│   ├── Header.tsx         # Navigation header
│   └── Footer.tsx         # Site footer
├── public/               # Static assets
└── ...config files
```
